## 1.0.0

* Initial release
* Added 11 animation widgets:
  - BlinkingAnimation - Blinking/pulsing opacity effect
  - BounceAnimation - Bouncing jump effect
  - RotateAnimation - Continuous rotation
  - SlideInAnimation - Slide in from any direction (with fromLeft, fromRight, fromTop, fromBottom constructors)
  - FadeInAnimation - Simple fade in effect
  - ScaleInAnimation - Zoom in/scale effect
  - WaveAnimation - Wave/floating effect
  - PulseAnimation - Pulsing scale effect
  - ShakeAnimation - Shake effect for errors
  - ShimmerAnimation - Shimmer loading effect
  - FadeSlideAnimation - Combined fade and slide animation
* All animations support customizable duration, curves, and delays
* Full documentation and examples included
